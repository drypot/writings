### Mac

맥이라면 brew 로 설치하는 것을 권장한다. (brew 가 먼저 설치되어 있어야한다.)

    $ brew install git

리포지터리 로그인 정보를 케쉬할 수 있으므로 해둔다.

    $ git config --global credential.helper osxkeychain

### Ubuntu

    $ apt-get install git-core

### Redhat

    $ yum install git-core
