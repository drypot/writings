빈하고 jsf 콤포넌트하고 연결은 잘 되는데,

필요할 때마다 컨트롤 파트 메서드 단위로 연결해야 하는데 복잡해 지면 장난 아닐 듯.

이 복잡도를 Managed Bean DI 로 해결하려고 했는가 봄. =,=

애초에 복잡하지 않게 되게 만들어야지 복잡하게 만들고 그걸 땜빵하려고 하다니. =,=
